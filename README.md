# dod.mag - 웹 매거진 프로젝트

일상 속 새로운 발견을 기록하는 감각적인 온라인 매거진

## 프로젝트 개요

**목적**: 자동화된 콘텐츠 생성 및 수익화를 위한 웹 매거진
**이름**: dod.mag
**기술 스택**: HTML, CSS, JavaScript (순수 바닐라)
**디자인**: Pretendard 폰트 기반 미니멀 디자인

## 주요 기능

### 카테고리
- 인사이트
- 신상품
- 이달의 타입
- 건강과 운동

### 관리자 기능
- **로그인**: 로고 좌측 투명 버튼 3번 클릭
- **비밀번호**: admin1991
- 글 작성, 수정, 삭제 권한

### 메인 캐러셀
- 최신 글 5개 자동 슬라이드쇼
- 5초 자동 전환
- 화살표 및 점 내비게이션

### 게시글 레이아웃
- 카테고리별 5개씩 가로 배치
- 이미지 → 제목 → 요약(2-3줄) → 날짜 순서
- 반응형 디자인 (PC: 5개, 태블릿: 3-4개, 모바일: 2개)

## 디자인 컨셉

- **폰트**: Pretendard Variable (제목 700-800, 본문 400-500)
- **배경**: #fafafa (따뜻한 오프화이트)
- **텍스트**: #1a1a1a (부드러운 검정)
- **여백**: 넓은 여백으로 숨 쉬는 레이아웃
- **이미지**: Unsplash 무료 이미지 사용 (상업적 이용 가능)

## 초기 샘플 게시글 (5개)

1. **인사이트** - 강한 도쿄 대지진 이후 2026년을 뜨겁게 할 5가지 라이프 트렌드
2. **인사이트** - 요즘 MZ는 아침 6시에 운동한다? 모닝 루틴의 진화
3. **인사이트** - AI가 만든 콘텐츠 vs 사람이 만든 콘텐츠, 2026년 승자는?
4. **신상품** - 올해의 IT 가젯 5선, 당신의 일상을 바꿀 제품들
5. **건강과 운동** - 운동 말고 움직임, 2026 헬스케어 키워드는 NEAT

## 로컬에서 실행하기

1. 브라우저에서 `index.html` 파일 열기
2. 로고 왼쪽 투명 버튼 3번 클릭 → 비밀번호 입력
3. 관리자 모드에서 글 작성/수정/삭제

**로컬 스토리지 초기화**:
```javascript
// 개발자도구 Console에서 실행
localStorage.clear()
```
