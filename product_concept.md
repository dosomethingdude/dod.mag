## EmailJS 설정 가이드

### 1단계: EmailJS 계정 생성
1. https://www.emailjs.com 접속
2. 회원가입 (무료 플랜: 월 200건 발송)

### 2단계: Email Service 연결
1. EmailJS 대시보드 > Email Services > Add New Service
2. Gmail, Outlook 등 선택 후 연결
3. **Service ID** 복사 (예: `service_xxxxxx`)

### 3단계: Email Template 생성
1. EmailJS 대시보드 > Email Templates > Create New Template
2. 아래 변수들을 템플릿에서 사용:
   - `{{to_email}}` - 수신자 이메일
   - `{{to_name}}` - 수신자 이름
   - `{{subject}}` - 제목
   - `{{message}}` - 본문 내용
   - `{{post_title}}` - 게시글 제목
   - `{{post_category}}` - 게시글 카테고리
   - `{{post_summary}}` - 게시글 요약
3. **Template ID** 복사 (예: `template_xxxxxx`)

### 4단계: Public Key 확인
1. EmailJS 대시보드 > Account > API Keys
2. **Public Key** 복사

### 5단계: 코드에 적용
`index.html`에서 `EMAILJS_CONFIG` 객체의 값 수정:
```javascript
const EMAILJS_CONFIG = {
    publicKey: 'YOUR_PUBLIC_KEY',      // 여기에 Public Key
    serviceId: 'YOUR_SERVICE_ID',      // 여기에 Service ID
    templateId: 'YOUR_TEMPLATE_ID'     // 여기에 Template ID
};
```

## 다국어 지원 안내

### 지원 언어
- **한국어 (KO)** - 기본 언어
- **영어 (EN)**
- **중국어 (中)**

### 사용 방법
- 헤더 우측의 언어 버튼(KO/EN/中) 클릭
- 선택한 언어로 UI 전체가 변환됨
- 선택한 언어 설정은 LocalStorage에 저장되어 재방문 시 유지

### 번역 범위
- 네비게이션 메뉴
- 카테고리명 및 설명
- 구독/로그인 모달
- 버튼 텍스트
- 안내 메시지

> 참고: 게시글 본문은 원문 그대로 표시됩니다. 콘텐츠 번역이 필요한 경우 각 언어별 게시글을 별도로 작성해야 합니다.
